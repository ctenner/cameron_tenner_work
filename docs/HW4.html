Penman/Combination: Free Water Surface Evaporation

Given our assumptions the Penman Combination Equation is as follows: \[E_0 = \frac{1000\Delta}{\rho_w \lambda_v \left[\Delta + \gamma \right]} \left[R_n\right] + \frac{\gamma}{\Delta + \gamma}\frac{6.43\left[1+0.536\overline{u_1} \right]}{\lambda_v} e_s\left[1 - RH\right]\] Known values:

  • Rn
  • T
  • u1
  • RH
  • density of water = 997 kg/m^3
  • gamma (psychrometric constant), = 0.066

We need to find:

  • e (vapor pressure), can find with T
  • es (saturation vapor pressure), can find with RH and e
  • latent heat of vaporization, can find with T
  • delta (sat vapor pres-temp gradient), can find with T and es
  • E0 (estimated potential evaporation), can find with all above
gamma = 0.066
rowater = 997

penman_combo <- april %>% mutate(
  e = 0.611 * exp(17.27*Ta/(237.3+ Ta)),
  es = e/RH,
  lambdav = 2.501 - (2.361 * (10^-3) * Ta),
  delta = 4098*es/((Ta+237.3)^2),
  E0 = (1000 * delta / (rowater*lambdav*(delta + gamma)) * RH) + ((gamma / (delta + gamma)) * ((6.43 * (1+(0.536 * Va))) / lambdav) * es * (1-RH))
)

Penman/Monteith

Given our assumptions the Penman Monteith Equation is as follows: \[E_{rc} = \frac{1000\Delta}{\rho_w \lambda_v \left[\Delta + \gamma^* \right]} \left[R_n\right] + \frac{\gamma}{\Delta + \gamma^*}\frac{900}{\left[T_a + 275\right]} \overline{u_1}e_s\left[1 - RH\right]\] Need to additionally find:

  • gamma_star, can be found with gamma and u1
  • Erc, can be found with all above info
penman_mont <- april %>% mutate(
  e = 0.611 * exp(17.27*Ta/(237.3+ Ta)),
  es = e/RH,
  lambdav = 2.501 - (2.361 * (10^-3) * Ta),
  delta = 4098*es/((Ta+237.3)^2),
  gamma_star = gamma * (1 + (0.33*Va)),
  Erc = (1000 * delta / (rowater*lambdav*(delta + gamma_star)) * RH) + ((gamma / (delta + gamma_star)) * (900/(Ta+275)) * Va * es * (1-RH))
)

Now we join the two datasets together

joined <- left_join(penman_combo, penman_mont, by = "Day") 
joined_gather <- joined %>% select(Day, E0, Erc) %>% gather(key = "Type", value = "E", -Day)

Now we can graph results

ggplot(joined_gather, aes(x=Day, y=E, fill = Type)) + geom_col(position = position_dodge(width = .7)) + scale_y_reverse() + labs(title = "1998 Potential Evaporation and Reference Crop Evaporation for Ciudad Obrégon, Mexico", y = "Evaporative Flux (mm / day)", x = "Day of April") + theme(plot.title = element_text(hjust = 0.5, size = 12))

P3. ET & Crop Coefficients

Pajaro <- read.csv("daily_eto_variance.csv") %>% 
  mutate(
    month = format(as.Date(Date, "%m/%d/%y"), "%b"),
    month_day = format(as.Date(Date, "%m/%d/%y"), "%m/%d")
  ) 

Pajaro$month_day <- as.Date(Pajaro$month_day, format = "%m/%d")

First we need to assign a stage to each date

Pajaro <- Pajaro %>% mutate(
    stage = case_when(
      month == "May" ~ "1",
      month_day > as.Date("5/31", "%m/%d") &  month_day < as.Date("7/11", "%m/%d") ~ "2",
      month_day > as.Date("2/29", "%m/%d") & month_day < as.Date("4/1", "%m/%d") ~ "4",
      month_day < as.Date("3/1", "%m/%d") ~ "3",
      month_day > as.Date("7/10", "%m/%d") ~ "3"#,
      #month == "Apr" ~ "Fallow"
    )
  )

To make the Kco interpolation easier, I’m reordering the dataset to go from day 1 stage 1 to day 30 fallow (May to April)

start <- Pajaro[214:366,]
end <- Pajaro[1:213,]

pajaro_growing_season <- rbind(start,end)

Now we need to make a vector that interpolates Kco over the growing season

Stage 1 We need to find Kco using the average Erc for stage 1

mean((pajaro_growing_season %>% filter(stage == 1))$Target_Year_Eto) * 25.4 #(inch to mm)

stage1 <- rep.int(.23, NROW(pajaro_growing_season %>% filter(stage == 1)))

stage3 <- rep.int(.95, NROW(pajaro_growing_season %>% filter(stage == 3)))

Stage 2 interpolate between Kco for 1 and Kco for 3 We will be using a straight line

stage2 <- seq(from = .23, to = .95, length.out = NROW(pajaro_growing_season %>% filter(stage == 2)))

Stage 4 interpolate between Kco for 3 and Kco after stage 4

stage4 <- seq(from = .95, to = .9, length.out = NROW(pajaro_growing_season %>% filter(stage == 4)))

Fallow

fallow <- rep.int(0, 30)

Now we can assign this vector to a “Kco” column, and calculate consumptive use by multiplying Kco and Eto.

allstages <- c(stage1, stage2, stage3, stage4, fallow)
pajaro_growing_season <- pajaro_growing_season %>% mutate(
  Kco = NA
)
pajaro_growing_season$Kco <- allstages

pajaro_growing_season <- pajaro_growing_season %>% mutate(
  consumptive_use = Kco * Target_Year_Eto * 25.4, 
  day_of_growing_season = row_number()
)
